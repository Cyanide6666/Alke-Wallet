### ¿Qué es Alke Wallet?
El objetivo de este proyecto "Alke Wallet" es desarrollar una aplicación de
billetera digital que permita a los usuarios gestionar sus activos financieros de
manera segura y conveniente, con un diseño femenino inspirado en el mundo de Sanrio :3 

### Elementos y lenguaje usados:
Se utilizó HTML5, CSS3, JavaScript, Bootstrap y JQuery

### Estructura del proyecto:

Alke-Wallet/

feature/login/ (Sistema de autenticación) 

feature/transacciones/ (Envío y recepción) 

feature/depositos/ (Gestión de depósitos)

main/  (Versión completa)
